## 巢狀迴圈 Nested Loop  

# 基本架構  

在迴圈裡面再塞迴圈  

例如輸出一個 $10 \times 10$ 的正方形 `*`  
當然單層的 `for` 迴圈就可以做到  

但如果改成雙層的 `for` 迴圈  

### code  

```python
for i in range(0,10,1):
    for j in range(0,10,1):
        print("*",end="")
    print("")
```

### output  

```
**********
**********
**********
**********
**********
**********
**********
**********
**********
**********
```

巢狀迴圈的運作順序，可以想成類似時鐘的運作  
分針就像內層迴圈，會先跑完  
外層迴圈，也就是時針才會走  

### code  

```python
for i in range(0,10,1):
    for j in range(0,i+1,1):
        print("*",end="")
    print("")
```

### output  

```
*
**
***
****
*****
******
*******
********
*********
**********
```

|       | 變數 `i` | 變數 `j` | 輸出 `*` 數量 |
|:-----:|:-----:|:-----:|:-----:|
| 第1圈 |   0   |   0 ~ x   |0|
| 第2圈 |   1   |   0 ~ 0   |1|
| 第3圈 |   2   |   0 ~ 1   |2|
| 第4圈 |   3   |   0 ~ 2   |3|
|...|...|...|...|
| 第10圈 |   9   |   0 ~ 8   |9|
